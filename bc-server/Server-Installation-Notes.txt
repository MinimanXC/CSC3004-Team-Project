----- SERVER -----

**Docker**

$ docker image rm bc-server
$ docker image rm syabz03/bc-server
$ docker build -t bc-server .
$ docker image tag bc-server:latest syabz03/bc-server:latest
$ docker image push syabz03/bc-server:latest

Apply settings for Persistent Storage on Docker Desktop 
* Navigate to Settings (Gear icon) > Resources > File Sharing, add directory "private/tmp/server-data"

**Kubernetes**

On Terminal, Execute the following commands

Create our Namespace 
$ kubectl create namespace bc-server

Create our Persistent Volume and Persistent Volume Claim
$ kubectl create -f persistent_volume.yml
$ kubectl create -f persistent_volume_claim.yml
$ kubectl get pv -o=jsonpath='{.spec.hostPath.path}' serverdisk -n bc-server

Create our Deployment Services
kubectl create -f bc-server.deployment.yml 
kubectl create -f bc-server.service.yml

Check on the Deployed Server 
$ kubectl get all -n bc-server

Check out Python prints!
$ kubectl logs pod/bc-server-ff6db6454-4w595 -n bc-server

e.g. $ kubectl logs pod/bc-server-ff6db6454-2k5xw -n bc-server

Persistent Data
$ kubectl exec -ti pod/bc-server-ff6db6454-2k5xw -n bc-server -- ls -la ../server-data

Kill the deployment
kubectl delete --all pods --namespace=bc-server
kubectl delete --all deployments --namespace=bc-server
kubectl delete --all services --namespace=bc-server

Remove Persistent Volume & Persistent Volume Claim
kubectl delete persistentvolumeclaim/serverclaim --namespace=bc-server
kubectl delete persistentvolume/serverdisk --namespace=bc-server --grace-period=0 --force
